ajAntiXray Documentation

Version: 2.0.5+ | Compatibility: 1.8.8 – 1.21.x

A high-performance, lightweight tool for detecting potential X-rayers on your Minecraft server by monitoring ore mining rates.

1. Key Features

  • Real-time Notifications: Instant alerts for staff when players exceed mining thresholds.
  • Flexible Tracking: Monitors any block type defined in the config.
  • Modern Messaging: Full MiniMessage support for colorful, interactive chat alerts.
  • Smart Exclusions: Integrates with Factions and supports Y-level height limits.
  • Discord Support: Built-in Webhook integration for remote monitoring.

2. Commands & Permissions

Command Permission Description
/ajaxr <player> ajaxr.check View mining stats for a specific player.
/ajaxr reload ajaxr.reload Reloads the configuration and messages.
N/A ajaxr.notify Receive alerts when someone is flagged.

3. Configuration Settings

The config.yml allows you to tune the sensitivity of the detection:

  • blocks-to-track: List of materials (e.g., DIAMOND_ORE, DEEPSLATE_DIAMOND_ORE).
  • notify-count: The amount of ores that triggers an alert.
  • check-interval: How many minutes before the "mined" counter resets.
  • ignore-above-y: Useful for ignoring ores found near the surface.
Important Note: This plugin is a detection tool, not an automated judge. Always spectate a player before taking disciplinary action, as lucky mining streaks or Silk Touch usage can cause false positives.
